# Shoplane
Shoplane E-Commerce Project
A fully responsive e-commerce web application designed to provide a seamless shopping experience. This project features a dynamic product catalog, category-based filtering, and a functional shopping cart system. It focuses on high performance, clean UI/UX, and a mobile-first approach to ensure accessibility across all devices.
